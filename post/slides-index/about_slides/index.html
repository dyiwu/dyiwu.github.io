Keyboard shortcuts

  • N, SPACE: Next slide
  • P: Previous slide
  • ←, H: Navigate left
  • →, L: Navigate right
  • ↑, K: Navigate up
  • ↓, J: Navigate down
  • Home: First slide
  • End: Last slide
  • B, .: Pause (Blackout)
  • F: Fullscreen
  • ESC, O: Slide overview / Escape from full-screen
  • S: Speaker notes view
  • ?: Show keyboard shortcuts
  • Ctrl + click: Zoom in. Repeat to zoom back out.
